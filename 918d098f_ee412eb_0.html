程序空间分配查看:
链接器根据ram.ld的规则, 进行代码和数据的空间分配, 编译后, 分配结果可以在Output\bin\map.txt中查看
如下图中, com区大小可以看到为34K.
如果需要增加com或bank区段名, 可以在ld文件中增加. 注意要放到对应的com区/bank区空间.
//----------------------------------------------------------------------------------
com区:
如上图中,com区位于0x20000~ (0x20000+34K) 的RAM空间内:
comm(rx)        : org = __comm_vma ,        len = __comm_ram_size   // 即 0x20000~ (0x20000+34K)
上面的"*"号, 表示任意名字, 如 AT(.com_text.timer) 也存放到AT(.com_text*)段中.

bank区 (也称为flash区):
如上图中,flash区在芯片内部映射地址为 0x10000000 ~ (0x10000000+512+768K)的区内.
flash(rx)       : org = __base + 512,       len = __max_flash_size
其它专用RAM映射,如aram/bram/dram/eram..., stack/data等, 可以在ld文件中查看.
